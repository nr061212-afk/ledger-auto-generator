# Ledger Auto Generator

銀行明細CSV・カード明細CSVから仕訳を自動生成するアプリです。

## 概要

会計事務所での記帳業務を効率化するために開発しました。  
銀行・カードの取引データとルールCSVをもとに仕訳を自動生成し、未判定データを分離して確認できます。

また、vendor_master による摘要の分類と、文字正規化（NFKC）により、表記ゆれを吸収した安定した判定を実現しています。

---

## 主な機能

- 銀行明細CSV・カード明細CSVの読み込み
- vendor_master による摘要分類（例：セブンイレブン → コンビニ）
- 文字正規化による表記ゆれ吸収（全角/半角・記号など）
- ルールCSVによる仕訳自動生成
- 未判定データの分離表示
- 件数サマリー表示（総件数 / OK / 未判定）
- 仕訳CSV・未判定CSVのダウンロード

---

## 技術スタック

- Python
- pandas
- Streamlit

---

## 使用方法

1. 銀行明細CSVまたはカード明細CSVをアップロード
2. ルールCSVをアップロード
3. 「仕訳を生成する」をクリック
4. 結果を確認しCSVをダウンロード

---

## 設計のポイント

description（元摘要）  
↓  
vendor_masterで分類（例：セブンイレブン → コンビニ）  
↓  
normalize_textで整形（記号削除・表記統一）  
↓  
rules.csvと照合  
↓  
仕訳生成  

- 意味の統一（分類）と文字の統一（正規化）を分離
- 未判定は理由付きで出力し、人間が判断できる設計

---

## 補足（開発用）
