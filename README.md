# Ledger Auto Generator

銀行明細CSVから仕訳を自動生成するStreamlitアプリです。

## 概要

会計事務所での記帳業務を効率化するために開発しました。
銀行取引データとルールCSVをもとに仕訳を自動生成し、未判定データは分離して表示します。
※現在も改善中のプロジェクトです。

## 主な機能

* 銀行明細CSVの読み込み
* ルールCSVによる仕訳自動生成
* 未判定データの分離表示
* 件数サマリー表示（総件数 / OK / 未判定）
* 仕訳CSVダウンロード

## 技術スタック

* Python
* pandas
* Streamlit

## 使用方法

1. 銀行明細CSVをアップロード
2. ルールCSVをアップロード
3. 「仕訳を生成する」をクリック
