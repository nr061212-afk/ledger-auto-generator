# 仕訳自動生成ツール

## 目的
摘要から勘定科目を自動判定し、仕訳データを生成する

## 入力CSV
- date: 取引日
- description: 摘要
- amount: 金額
- type: 入出金区分

## ルールCSV
- priority: 優先順位
- keyword: キーワード
- debit_account: 借方
- credit_account: 貸方

## 出力CSV
- date
- debit_account
- credit_account
- amount
- description
- status

## 判定ルール
- keywordが含まれていたら判定
- priorityで優先順位制御
- 該当なしは未判定
